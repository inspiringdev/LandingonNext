# LandingonNext

A modern frontend web application for a specialty coffee shop. This project features a responsive, mobile-first design with smooth scroll animations and a glassmorphic UI.

## Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React

## Getting Started

To run this project locally on your machine, follow these steps:

1. Clone the repository to your local machine.
2. Open your terminal and navigate into the project directory.
3. Install the required dependencies:
